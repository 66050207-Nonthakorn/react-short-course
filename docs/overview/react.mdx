# React

React (หรืออาจจะเคยได้ยินในชื่อ React.js) คือ **Library** ในการพัฒนาเว็บไซต์ Frontend พัฒนาขึ้นโดย
[Meta](https://en.wikipedia.org/wiki/Meta_Platforms) หรือที่เรารู้จักในชื่อ Facebook

## Solution

จากหน้าที่แล้วเราลองคิดดูว่า ทำไมเราต้องส่งมาทีละครั้งในเมื่อเราสามารถส่งมารวดเดียวเลยทุกหน้าแล้วเวลาอยากเปิดหน้าไหนก็เปิด ก็ไม่ต้องไปขอ Google หลายๆรอบ
แต่หลักการคิดนี้ก็ไม่ง่ายเสมอไปเพราะถ้าเว็บไซต์มีมากกว่า 10 หน้าหรือเยอะกว่านั้นการขอไปแต่ละครั้งก็คงช้าในการส่ง

ในตอนที่เราขอ Server ไปนั้นสิ่งที่ React ทำคือเราจะส่งข้อความ **HTML** ไปและอีกอย่างนึงเรียกว่า **JS Bundle** (Javascript Bundle) แค่ครั้งเดียว

ให้เปรียบเทียบการทำงานก็คงเหมือนเพื่อนจะให้บ้านกับเรา แต่เพื่อนก็คงส่งบ้านผ่าน EMS ให้เราไม่ได้เพราะมันใหญ่เกิน
ดังนั้นเพื่อนก็ส่งโครงบ้านกับช่างซึ่งมีหน้าที่สร้างบ้านให้เรามาแทน

- บ้าน = **หน้าเว็บ**
- โครงบ้าน = **HTML**
- ช่าง = **JS Bundle**
